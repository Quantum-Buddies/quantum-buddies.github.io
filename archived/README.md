# Archived Assets

These files have been removed from the live site but kept for potential reuse.

## surreal-scene.js
**Three.js wireframe terrain with scroll-reactive moods.**

Features:
- Simplex noise-driven wireframe terrain (red/crimson/purple net wave)
- Starfield with twinkle animation
- Dual orbital rings
- Scroll-reactive mood system (color, amplitude, density shift per section)
- Mouse-driven ripple interaction on terrain
- Smooth mood transitions via lerping

Dependencies: Three.js r128 (CDN)

To reuse: include Three.js and this script, plus a `<div id="canvas-container"></div>` in your HTML.
